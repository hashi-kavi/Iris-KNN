# Iris Classification using KNN

## Overview
This project applies the K-Nearest Neighbors (KNN) algorithm to classify
Iris flower species using Python and scikit-learn.

## Dataset
- Iris Dataset (CSV)
- Features: Sepal & Petal measurements
- Target: Flower species

## Method
- Standardization using StandardScaler
- Train-test split (80/20)
- KNN classification
- Evaluation using accuracy and confusion matrix
- Performance tested for K = 1 to 19

## Results
- Best performance observed at K = 8

## Technologies
- Python
- NumPy
- Pandas
- scikit-learn
